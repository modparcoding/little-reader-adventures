# Little Reader Adventures

A UK-aligned, gamified, iPad-friendly reading app for a 5-year-old, built as a single-page web app.

## Features
- Phonics scope with multiple levels:
  - Single sounds
  - Short vowels and final sounds
  - Common digraphs
  - Blend and blend clusters
- Word Builder levels with progressive words
- High-frequency words in expanding bands
- Tiny stories with comprehension questions
- Gamification: XP, coins, streak, missions, and badge rewards
- Daily quest system with rewards
- Multiple visual themes
- Parent panel with goals + progress summary

## Files
- `index.html`
- `styles.css`
- `script.js`

## How to run on iPad
1. Open this folder.
2. Start a local server (recommended): `python -m http.server 8080`
3. Open `http://<your-computer-ip>:8080/index.html` in iPad Safari.
4. Add to Home Screen for full-screen use.
